Overview

Built a full-stack portal using React, Tailwind CSS, and FastAPI for industrial vision systems. Designed for clients and internal teams to manage, monitor, and analyze real-time data streams.

Core Components

  • Dataset Management: Upload, tag, and organize image datasets for training and analysis.
  • System Management: Monitor camera and server health, logs, and performance metrics in real time.
  • Model Management: Deploy, version, and evaluate AI/ML models with performance dashboards.

Key Features

  • Dynamic filtering and search for quick data access
  • Real-time statistics and alerts using WebSocket integrations
  • Image gallery with defect overlays and export options
  • Secure JWT-based authentication and role-based permissions

Architecture & Tech Stack

  • Front-End: React, Tailwind CSS, Ant Design components
  • Back-End: FastAPI, PostgreSQL, SQLAlchemy
  • Real-Time: WebSockets for live data updates
  • Deployment: Docker Compose, CI/CD pipelines

User Interface & UX

Designed in dark mode with a grey base and light blue accents for reduced eye strain and intuitive navigation.

Future Plans

  • Client-specific dashboards and custom reporting
  • Role-based access controls and audit logging
  • Integration with external data sources (e.g., PLCs, ERP)
