Music Uploader

Streamlined Flask service to download tracks from SoundCloud/YouTube and add them to your Apple Music library.

← Back to Home

Overview

A personal Flask-based web service using Python and AppleScript. Run a local server on macOS, access from any device, paste a SoundCloud or YouTube URL, and automatically download and upload the track to your Apple Music library.

Project Structure

  • main.py: Core Flask application and download logic
  • config.yaml: Application settings
  • templates/: Jinja2 HTML templates for the UI
  • static/: CSS & JavaScript assets
  • script.applescript: Automates Apple Music upload on macOS
  • requirements.txt: Python dependencies

Installation & Usage

  1. Clone: git clone https://github.com/BaileyHelfer/upload_amusic.git
  2. Create & activate venv:
    python3 -m venv venv && source venv/bin/activate
  3. Install deps: pip install -r requirements.txt
  4. Configure config.yaml
  5. Run: python main.py
  6. Open http://localhost:5000 in browser or phone

Challenges

Apple Music does not provide an official API for automating library uploads. The only way to add tracks programmatically is via macOS AppleScript when the user is already signed into the Music app—an imperfect but effective workaround.

Inspiration

I wanted a simpler way to add unreleased or streamed music to my Apple Music library without manually downloading and dragging files. This tool streamlines that process directly from SoundCloud or YouTube links.
